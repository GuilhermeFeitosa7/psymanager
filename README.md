PsycManager: Gestão Clínica para Psicólogos

Este é um projeto robusto desenvolvido para facilitar a rotina de profissionais de psicologia, 
automatizando desde o agendamento até a emissão de documentos clínicos, 
sempre prezando pela segurança dos dados e organização profissional.


Funcionalidades PrincipaisGestão de Pacientes: 

Prontuário digital completo com histórico de sessões.Agenda Inteligente: Controle de dias, horários e status de consultas.
Notificações Automáticas: Envio de confirmação de consulta via WhatsApp (integração via API).
Relatórios & Documentos: Geração automática de atestados e relatórios semanais em PDF.
Segurança (LGPD): Estrutura pensada para a proteção de dados sensíveis de saúde.


Stack Tecnológica

Camada					Tecnologia

Linguagem Principal		Java 17+
Framework Backend		Spring Boot (REST API)
Persistência de Dados	Spring Data JPA / Hibernate
Banco de Dados			PostgreSQL
Geração de Documentos	iText
Versionamento			Git & GitHub



Arquitetura e Design
O projeto segue os princípios do SOLID e a arquitetura em camadas (Controller, Service, Repository), garantindo que o código seja testável 
e de fácil manutenção.
